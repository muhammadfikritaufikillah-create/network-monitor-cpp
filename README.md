# Network Monitor C++

A command-line network monitoring tool built in C++ using POSIX socket programming.

## Features
- **Port Scanner** — Scan custom port range on any target IP
- **Quick Scan** — Instantly scan top 20 most common ports
- **Host Discovery** — Detect active hosts in a local network range
- **Auto Logging** — Automatically saves scan results to .txt files

## Build & Run
```bash
g++ network_monitor.cpp -o network_monitor
./network_monitor
```

## Requirements
- macOS or Linux
- g++ compiler
